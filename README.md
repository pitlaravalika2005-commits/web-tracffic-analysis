## 📌 Overview

This project analyzes website traffic data to understand user behavior. It focuses on key metrics such as page views, session duration, bounce rate, and traffic sources to identify patterns and improve engagement.

---

## 🎯 Objectives

* Analyze user activity
* Identify traffic sources
* Evaluate engagement metrics
* Visualize traffic patterns

---

## 📂 Project Files

* `traffic_data.csv` → Dataset
* `analysis.py` → Python script
* `README.md` → Documentation

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib

---

## ⚙️ Workflow

1. Load dataset
2. Calculate key metrics
3. Group data by traffic source
4. Analyze user engagement
5. Visualize traffic distribution

---

## ▶️ How to Run

pip install pandas matplotlib
python analysis.py

---

## 📊 Output

* Average page views
* Session duration analysis
* Bounce rate insights
* Traffic source graph

---

## 🏁 Conclusion

Organic traffic shows higher engagement compared to ads and social media. Users from organic sources tend to spend more time and view more pages.
